다이어그램이나 아키텍처를 그릴 때 ASCII art 대신 mermaid를 사용할 것.

**Why:** 사용자가 직접 요청. 렌더링 가능한 형태를 선호함.
**How to apply:** 학습자료에서 flowchart, sequence diagram, gantt 등 시각 자료가 필요할 때 항상 mermaid 코드블록 사용. style fill 등 커스텀 색상 지정은 하지 않는다 (대비 문제로 가독성이 떨어짐).
